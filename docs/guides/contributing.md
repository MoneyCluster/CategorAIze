---
title: Как вносить изменения
---

1. Добавьте/обновите Markdown в `docs/`
2. Для LikeC4 — измените модель в `docs/_likec4/*.c4` и проверьте локально (см. [Локальная разработка](./local-setup.md))
3. Локально проверьте сайт: `python3 -m mkdocs serve`
4. Commit + push — CI соберёт и опубликует

Подробнее: [Локальная разработка документации](./local-setup.md)
