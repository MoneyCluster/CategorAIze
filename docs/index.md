# CategorAIze

Система персонализированной классификации финансовых трат пользователей с использованием машинного обучения.

## О проекте

CategorAIze помогает автоматизировать категоризацию финансовых трат пользователей. В отличие от стандартных решений, система использует персонализированные ML-модели для каждого пользователя, что позволяет учитывать индивидуальные категории и привычки.

### Основные возможности

- **Персонализированная классификация**: Каждый пользователь получает собственную модель, обученную на его данных
- **Гибкие категории**: Пользователи определяют свои категории трат
- **Обратная связь**: Система обучается на корректировках пользователя
- **Быстрая работа**: Время отклика API ≤ 200 мс

### Архитектура

Система состоит из двух основных компонентов:

1. **Общий эмбеддер** (`sentence-transformers`) — преобразует текстовые описания трат в векторы
2. **Персональные классификаторы** (scikit-learn) — один для каждого пользователя

Подробнее об архитектуре см. в [ADR-001: Персонализированная классификация трат](adr/001-personalized-classification.md).

## Быстрый старт

Для локальной разработки документации:

```bash
./scripts/serve-docs.sh
```

Скрипт автоматически:
- Соберёт LikeC4 диаграммы (если установлен Node.js)
- Установит зависимости MkDocs (если нужно)
- Запустит локальный сервер на http://127.0.0.1:8000
